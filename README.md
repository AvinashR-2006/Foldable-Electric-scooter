# Foldable Electric Scooter

A compact foldable electric scooter concept focused on mechanical packaging, portability, and product-oriented engineering.

## Overview

This repository documents the concept, design intent, and engineering direction of my foldable electric scooter project. The current repository is documentation-first because the local workspace contains project visuals but not a full CAD or electronics source package.

## Focus Areas

- Foldable frame concept
- Compact mechanical layout
- Rider portability
- Controlled acceleration and electric drive integration
- Product-style design thinking

## Current Status

This repository is a portfolio-style record of the project. Additional CAD files, controller details, and manufacturing drawings can be added later if they become available.
